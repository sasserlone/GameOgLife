Conway's Spiel des Lebens

Generation: 0

Regeln

  • Eine lebende Zelle mit weniger als 2 lebenden Nachbarn stirbt (Unterbevölkerung).
  • Eine lebende Zelle mit 2 oder 3 lebenden Nachbarn überlebt.
  • Eine lebende Zelle mit mehr als 3 lebenden Nachbarn stirbt (Überbevölkerung).
  • Eine tote Zelle mit genau 3 lebenden Nachbarn wird lebendig (Fortpflanzung).

Klicke auf eine Zelle, um sie umzuschalten.
